TRex

An explanation enabled reasoner for Ontology Learning

TRex - Terminological Reasoning with Explanations

This is the homepage of the TRex reasoner. TRex is a reasoner for a subset of OWL2 and particularly suited to compute explanations for incoherences in ontologies generated by means of ontology learning.

Getting TRex

The TRex library JAR containing all dependencies is available for download. If you use maven for managing your dependencies, you can also add the following dependency to your POM

<dependency>
  <groupId>de.krkm.trex</groupId>
  <artifactId>trex</artifactId>
  <version>1.0.1</version>
</dependency>

and add the following repository to your projects repositories specification

<repository>
  <id>dws-unima</id>
  <url>https://breda.informatik.uni-mannheim.de/nexus/content/groups/public/</url>
</repository>

For documentation see the README, the API documentation and the example class.

Publications

Technical Report

The technical report is now available for download. It contains the full proof of incoherence completeness as advertised in the original paper.

Datasets

The ontologies used in the experiments are available for download. The full ontologies are the originally learned ontologies while the filtered ontologies are the same ontologies but filtered to only contain axioms supported by TRex. Finally, the RR2013 data set contains 11 subsets of the largest filtered ontology as used for the experiments in the technical report.
